## 一.Ubuntu系统直接安装脚本<br>
>如果您是服务器或者自备了`ubuntu` 请直接使用下面的命令<br>

```
apt update
```
```
apt install curl -y
```
```
bash <(curl -sL https://gitee.com/Wind-is-so-strong/yz/raw/master/install.sh)
```

<li>首次启动脚本似乎会自动关闭，再执行一遍就好了喵<li>
